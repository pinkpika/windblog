Death note 彌海砂

發表於9月 7 2019|更新於6月 4 2022|Cosplay

Death note 對我也是個蠻有意義的作品，學生中二時候意外接觸到漫畫，
我非常少看邏輯類的作品其實，但這部有趣的題材跟角色個性瞬間打到我
不管是心機又會轉變懂得看場合變化及利用他人的月、可愛怪異但又聰明忠於自我的L
或是愛的卡慘死的為了救命恩人單方面自認為戀人並奉獻的海砂 我都很喜歡

這次也如願完成懷舊夢，真心感恩～前言廢談完畢

想到那句經典的台詞，現在還是忘不了

這世上沒有天堂和地獄，生前不管做任何事而死的人都會去同一個地方，死亡是平等的。

也難怪月而言，殺這麼多人不痛不癢(我沒在怕的拉(X






偷偷說一下，其實縮圖不太愛放臉，有時候覺得自己好崩 (那你是在拍什麼(拍夢想的(ㄎ
